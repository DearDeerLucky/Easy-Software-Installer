# Easy software installer
> 用于通过脚本自动下载安装包并自动执行安装程序  
> 装机必备

## 介绍
主要通过python脚本来执行下载和执行安装程序

## 准备工作
1. 以文件形式下载本项目
2. 安装 [python](https://www.python.org/ftp/python/3.10.8/python-3.10.8-macos11.pkg)

## Windows
打开PowerShell/CMD客户端
运行脚本
``` shell
pip install tqdm
sudo python easy_software_installer.py
```
